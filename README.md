# E-commerce Product Analytics Project

## Project Overview
End-to-end analytics and product improvement project focused on understanding customer behavior, operational performance, and business insights using real-world e-commerce data.

## Business Problem
The organization faces inconsistent revenue, delivery delays, and declining customer retention. The project aims to identify root causes and recommend improvements using data analysis.

## Tools Used
- SQL (SQLite)
- Power BI
- Git & GitHub
- Business Analysis Documentation
- Project Management Planning

## Dataset
Brazilian E-commerce Public Dataset (Olist)

## Project Structure
- data/ → datasets
- sql/ → SQL queries
- analysis/ → business insights & questions
- dashboards/ → Power BI dashboards
- brd/ → business documentation
- management/ → project planning
- presentation/ → final case study
